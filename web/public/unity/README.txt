Place your Unity WebGL build files in this folder.
Recommended structure:
- public/unity/Build/MortarGame.loader.js
- public/unity/Build/MortarGame.data.unityweb
- public/unity/Build/MortarGame.framework.js.unityweb
- public/unity/Build/MortarGame.wasm.unityweb
- public/unity/StreamingAssets/ (optional, if used)

You can change the base name in src/app/game/page.tsx (buildBase constant) to match your build.
